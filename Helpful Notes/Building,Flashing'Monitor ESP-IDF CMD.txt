NOTE: Remember, need to connect GPIO0 to ground when booting if you want to flash to device!

1: cd INTO project folder.
2: >idf.py set-target esp32
3: >idf.py menuconfig
   3.1: [Change as required for project].
   3.2: Save and exit.
4: >idf.py build
5: >idf.py -p [PORT GOES HERE] flash
6: >idf.py monitor

NOTE: If nothing is displayed just press the reset button, if still nothing is displayed you have messed up.
